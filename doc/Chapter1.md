# 第1节：HTTP中间人代理实现

想实现HTTPS的中间人代理，我们先定个**小目标**，先实现一个**HTTP**的中间人代理。

HTTP 和 HTTPS的区别
-------
HTTP协议是互联网应用最广泛的一种网络协议，传输的内容是以明文方式进行。由于传输的是明文，HTTP本身没有任何安全保障。HTTPS的主要思想就是在不安全的网络上创建一个安全信道，这里的**S**是指：SSL/TLS就是这一层安全通道。
